# macの初期設定

## macの設定を変えたもの
### 自動翻訳の解除
`設定 > キーボード > 入力ソース > ライブ変換`

### caps lockをcontrolに
`設定 > キーボード > 入力ソース > 修飾キー`
https://support.apple.com/ja-jp/guide/mac-help/mchlp1011/mac

### docsを隠す
`設定 > docs > 自動的に表示非表示`

### 左下でスクリーンセーバー
`デスクトップとスクリーンセーバー > ホットコーナー`

### 同じアプリケーションのウィンドウを切り替える
`option + tab`にする
https://dev.classmethod.jp/articles/mac-next-window-shortcut/
